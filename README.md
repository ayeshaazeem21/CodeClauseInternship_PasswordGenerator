# CodeClauseInternship_PasswordGenerator

Aim -
Create a password generator that generates strong and random passwords.

Description-
Build a program that allows users to specify password requirements (length,
inclusion of numbers/symbols, etc.) and generates a secure password.

Code-
The code takes the length of the password from the user and asks whether the user wants numbers and special symobls in the password. Acoording to the user the password is randomly generated using a for loop and the output is printed on the screen.
